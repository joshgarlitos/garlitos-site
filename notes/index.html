generide

Last updated July 2026

A Python tool for generating roller coaster track files that load directly into OpenRCT2. Phase 1 is done: a real .td6 file round-trips through Python byte-for-byte. Phase 2 is track segment geometry. The long-term goal is a genetic algorithm that evolves coasters players actually want to ride.
