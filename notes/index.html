generide: Evolving Playable Roller Coasters for OpenRCT2

Last updated: July 12, 2026

A Python tool that reads, validates, generates, and evolves RollerCoaster Tycoon 2 track designs. Reverse-engineering a 1999 binary format, validating 3D track geometry, and evolving rides that run in the game.
